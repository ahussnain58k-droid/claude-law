# Obligations Routing Standard

Every obligation must include:

- source section
- obligated party
- action required
- trigger/deadline
- owner
- risk if missed
- routing destination

Common routes:

- privacy-impact-assessment
- dpa-review
- dsar-response
- ai-use-case-triage
- ai-risk-assessment
- vendor-ai-review
- confidentiality-risk-checker
- privilege-risk-checker
- citation-checker
- human-review-gate
